# Strict Pull Up

## ¿Qué es un Strict Pull Up?
El **Strict Pull Up** es un ejercicio gimnástico de peso corporal que consiste en levantar el cuerpo hasta que la barbilla supere la barra. Este movimiento desarrolla fuerza en la espalda, los bíceps y el core.

## ¿Cómo se hace un Strict Pull Up?
1. **Posición inicial**: Cuélgate de una barra con las manos en pronación (palmas hacia adelante), separadas a una distancia ligeramente mayor que el ancho de los hombros. Activa el core y los músculos de la espalda.
2. **Ejecución**: Tira del cuerpo hacia arriba contrayendo los músculos de la espalda y los bíceps, llevando la barbilla por encima de la barra.
3. **Finalización**: Baja el cuerpo de manera controlada hasta que los brazos queden completamente extendidos.

## ¿Cómo se puede escalar un Strict Pull Up?
- **Band-assisted Strict Pull Up**: Usa una banda elástica para asistencia, colocando los pies o rodillas en la banda.
- **Jumping Pull Up**: Realiza el movimiento utilizando un salto para ganar altura.
- **Negative Pull Up**: Concéntrate en bajar controladamente desde la posición superior.
