# Single Under

## ¿Qué es un Single Under?
El **Single Under** es el salto básico con cuerda, donde esta pasa una vez bajo los pies por cada salto. Se trabaja la coordinación y la resistencia y es perfecto para principiantes y como calentamiento.

<YouTube id="hCuXYrTOMxI" class:list={[ 'rounded']}/>

## ¿Cómo se hace un Single Under?
1. **Posición inicial**: Sostén la cuerda con las manos a los lados del cuerpo y los pies juntos.
2. **Ejecución**: Gira la cuerda suavemente mientras saltas justo lo necesario para que pase bajo los pies.
3. **Finalización**: Aterriza suavemente y repite a un ritmo constante.

Los codos deben estar cerca del cuerpo y recuerda usar las muñecas para girar la cuerda.

## ¿Cómo se puede escalar un Single Under?
- Practica sin cuerda para dominar el salto.
- Realiza saltos con un ritmo más lento para ajustar la coordinación.
