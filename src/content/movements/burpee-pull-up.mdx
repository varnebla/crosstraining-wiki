# Burpee Pull Up

## ¿Qué es un Burpee Pull Up?
El **Burpee Pull Up** combina dos movimientos: el Burpee y el Pull-up, trabajando tanto el sistema cardiovascular como la fuerza de la parte superior del cuerpo.

<YouTube id="WoZolg3iaZ0" class:list={[ 'rounded']}/>

## ¿Cómo se hace un Burpee Pull Up?
1. **Posición inicial**: Ponte de pie frente a una barra de Pull-up.
2. **Ejecución**: Realiza un Burpee (baja al suelo, haz una flexión, levántate y salta hacia arriba).
3. **Finalización**: Al saltar, agarra la barra y realiza un Pull-up completo. Baja controladamente y repite.

Para hacer el movimiento más fluido, aprovecha el salto del burpee.

## ¿Cómo se puede escalar un Burpee Pull Up?
- **Burpee + Jump**: Realiza solo el salto hacia la barra sin el Pull-up.
- **Jumping Pull-up**: Salta hacia la barra y utiliza el impulso para completar el Pull-up.
- **Band-assisted Pull-up**: Usa una banda elástica para asistencia durante la tracción.
