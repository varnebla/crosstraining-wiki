# Inchworm

## ¿Qué es un Inchworm?
El **Inchworm** es un ejercicio dinámico que combina movilidad y fuerza, en el que se practica el caminar con las manos. Se utiliza comúnmente como calentamiento o para mejorar la flexibilidad y la activación muscular.

<YouTube id="zfXLz6EHd7U" class:list={[ 'rounded']}/>

## ¿Cómo se hace un Inchworm?
1. **Posición inicial**: Ponte de pie con los pies al ancho de las caderas.
2. **Ejecución**: Inclínate hacia adelante, llevando las manos al suelo, y camina con ellas hacia una posición de plancha.
3. **Finalización**: Camina con los pies hacia las manos para volver a la posición inicial.

Recuerda mantener el core activado y evita arquear la espalda durante el movimiento.

## ¿Cómo se puede escalar un Inchworm?
- **Short Inchworm**: Realiza el movimiento sin llegar completamente a la posición de plancha.
- **Bent-knee Inchworm**: Dobla ligeramente las rodillas para facilitar el alcance del suelo.
- **Stationary Plank Walk**: Haz un movimiento similar sin desplazarte hacia adelante.
