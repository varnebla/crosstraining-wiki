# L-Sit

## ¿Qué es un L-Sit?
El **L-Sit** es un ejercicio isométrico avanzado donde el cuerpo forma una "L" mientras se sostiene en paralelas, barra o anillas. Es ideal para fortalecer el core y mejorar la estabilidad.

<YouTube id="WHi1bvZLwlw" class:list={[ 'rounded']}/>

## ¿Cómo se hace un L-Sit?
1. **Posición inicial**: Sube a unas paralelas, barra o anillas, y mantén los brazos extendidos con los hombros activos.
2. **Ejecución**: Eleva las piernas rectas frente al cuerpo hasta formar un ángulo de 90 grados. Los hombros deben estar activos y el core bien apretado para evitar colapsar la postura.
3. **Finalización**: Mantén la posición durante el tiempo deseado y baja controladamente.

## ¿Cómo se puede escalar un L-Sit?
- **Tuck Hold**: Mantén las rodillas dobladas hacia el pecho.
- **L-Sit Kick-out**: Extiende una pierna a la vez mientras sostienes la posición.
