# Push Press

## ¿Qué es un Push Press?
El **push press** es un movimiento en el que se utiliza un ligero impulso de piernas para empujar la barra sobre la cabeza. Es una combinación de fuerza de piernas y de hombros.

<YouTube id="iaBVSJm78ko" class:list={[ 'rounded']}/>

## ¿Cómo se hace un Push Press?
1. Coloca la barra en los hombros, con las manos un poco más anchas que los hombros.
2. Flexiona ligeramente las rodillas para tomar impulso.
3. Extiende rápidamente las piernas mientras empujas la barra por encima de la cabeza.
4. Baja la barra de forma controlada hasta los hombros.


## ¿Cómo se puede escalar un Push Press?
- **Dumbbell Push Press**: Usa mancuernas en lugar de la barra.
- **Strict Press**: Realiza el movimiento sin usar el impulso de las piernas.
- **Push Jerk**: Si el peso es más alto, utiliza el push jerk para lograr un impulso adicional.
