# Cluster

## ¿Qué es un Cluster?
El **Cluster** es un movimiento complejo que combina un Squat Clean con un Thruster. Este ejercicio es ideal para trabajar todo el cuerpo, desarrollando fuerza, potencia y resistencia.

## ¿Cómo se hace un Cluster?
1. **Posición inicial**: Coloca una barra en el suelo y adopta una postura de Squat Clean (pies a la altura de los hombros, espalda recta, manos al ancho de los hombros).
2. **Ejecución**: Realiza un Squat Clean, llevando la barra desde el suelo hasta la posición de rack frontal mientras bajas en sentadilla profunda.
3. **Finalización**: Desde la sentadilla, realiza un Thruster impulsando la barra hacia arriba hasta que los brazos queden extendidos. Baja la barra controladamente hasta el suelo y repite.

Mantén el core activado durante todo el movimiento para proteger la zona lumbar.

## ¿Cómo se puede escalar un Cluster?
- **Hang Squat Clean + Press**: Divide el movimiento en dos partes para enfocarte en la técnica.
- **Dumbbell Cluster**: Usa mancuernas en lugar de la barra para facilitar el control.
- **Light-weight Cluster**: Practica con menos peso para dominar el movimiento.
