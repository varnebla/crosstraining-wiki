# Push Jerk

## ¿Qué es un Push Jerk?
El **push jerk** es un movimiento explosivo en el que se utiliza una extensión de piernas para empujar la barra por encima de la cabeza y luego se flexionan las rodillas para recibirla en posición de bloqueo.

## ¿Cómo se hace un Push Jerk?
1. Coloca la barra en los hombros, con las manos a la altura de los hombros.
2. Flexiona las rodillas ligeramente y realiza un impulso hacia arriba.
3. Mientras la barra se eleva, flexiona las rodillas nuevamente para recibir la barra sobre la cabeza con los brazos extendidos.
4. Extiende las piernas para completar el movimiento.

## ¿Cómo se puede escalar un Push Jerk?
- **Push Press**: Realiza el movimiento sin la segunda flexión de rodillas.
- **Dumbbell Push Jerk**: Usa mancuernas en lugar de la barra.
- **Strict Press**: Realiza el movimiento sin usar las piernas para un mayor enfoque en los hombros.
