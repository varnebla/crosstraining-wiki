# Handstand Walk

## ¿Qué es un Handstand Walk?
El **Handstand Walk** es un ejercicio gimnástico avanzado que consiste en caminar usando las manos mientras el cuerpo está completamente invertido. Este movimiento mejora el equilibrio, la fuerza del core y el control corporal.

## ¿Cómo se hace un Handstand Walk?
1. **Posición inicial**: Realiza una invertida hasta una posición de pino libre con las manos en el suelo, activando el core para mantener el equilibrio.
2. **Ejecución**: Deja que tu peso te lleve hacia delante y mueve una mano hacia adelante mientras mantienes el control del cuerpo, alternando las manos para avanzar.
3. **Finalización**: Baja controladamente cuando completes la distancia deseada o pierdas el equilibrio.

## ¿Cómo se puede escalar un Handstand Walk?
- **Wall Walk**: Practica caminar hacia la pared para desarrollar control en la posición invertida.
- **Shoulder Taps**: Realiza toques de hombros mientras estás apoyado en una pared.
- **Kick-up Hold**: Trabaja en mantener el equilibrio en la posición de pino sin caminar.
